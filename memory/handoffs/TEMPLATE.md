# Handoff template

Copy this file, rename it `YYYY-MM-DD-short-topic.md`, and fill it in when a session ends mid-thread. Delete this instruction line in the copy.

## Date

DD Month YYYY

## What was in progress

One or two sentences: what the thread was actually trying to get done.

## What's done

What's actually finished and doesn't need revisiting.

## What's still open

The specific next step, concrete enough that a cold session (any model) could pick it up without asking "what did you mean."

## Anything the next session should NOT redo

Decisions already made, options already ruled out, so the next session doesn't re-litigate settled ground.
